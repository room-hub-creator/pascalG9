# Pascal's Binomial Calculator

A high-performance, academic-grade web application for generating Pascal's Triangle and computing Binomial Coefficients. Designed for the **MA80443** project.

## Features

- **Pascal's Triangle Visualizer**: Interactive generation of Pascal's Triangle with symmetric layout.
- **Binomial Coefficient Calculator**: Compute large $(n, r)$ values efficiently.
- **AI Assistant "KAMARAMPAKA"**: Integrated mathematical assistant powered by Groq (Llama 3) for step-by-step derivations.
- **Modern UI/UX**: Professional dashboard with glassmorphism, responsive design, and automatic theme support.
- **Edge Computing**: High-performance calculations using optimized algorithms.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **AI Integration**: Groq SDK
- **Testing**: Vitest

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Academic Context

This project fulfills the requirements for the MA80443 academic module, focusing on memory-efficient data structures and high-performance computation of mathematical triangles.
